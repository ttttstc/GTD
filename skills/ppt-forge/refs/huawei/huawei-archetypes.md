# 华为典型页型模板

> 华为 PPT 的典型页型结构参考。

## 页型总览

| 页型 | 密度 | 模块数 | 复杂度 | 适用场景 |
|------|------|--------|--------|---------|
| 战略总览 | 高 | 8-12 | 高 | 全局视图、战略分析 |
| 汇报页 | 中高 | 4-6 | 中 | 季度汇报、KPI 展示 |
| 架构页 | 高 | 6-10 | 高 | 系统架构、方案设计 |
| 对比分析 | 中高 | 4-8 | 中 | 方案对比、选型分析 |
| 发布会结论 | 低 | 2-4 | 低 | 产品发布、对外演讲 |

## 1. 战略总览页

### 特征
- 最高密度，信息量最大
- 多模块并列，展示全局视图
- 通常是封面后的第一页或总结页

### 布局结构

```
┌─────────────────────────────────────────────────────┐
│  [L1] 页面主标题                              [日期] │
├─────────────────────────────────────────────────────┤
│ ┌─────────┐ ┌─────────┐ ┌─────────┐ ┌─────────┐   │
│ │ KPI 1   │ │ KPI 2   │ │ KPI 3   │ │ KPI 4   │   │
│ └─────────┘ └─────────┘ └─────────┘ └─────────┘   │
├─────────────────────────────────────────────────────┤
│ [模块A]     │ [模块B]     │ [模块C]     │ [模块D]    │
├─────────────────────────────────────────────────────┤
│ [模块E]     │ [模块F]     │ [模块G]     │ [模块H]    │
├─────────────────────────────────────────────────────┤
│ ┌─────────────────────────────────────────────────┐│
│ │                 流程图 / 总结                    ││
│ └─────────────────────────────────────────────────┘│
└─────────────────────────────────────────────────────┘
```

### CSS 模板片段

```html
<div class="page-strategy">
  <header class="page-header">
    <h1 class="title">战略总览</h1>
    <span class="date">2026.03</span>
  </header>

  <div class="kpi-row">
    <div class="kpi-card">
      <span class="kpi-number">85%</span>
      <span class="kpi-label">目标完成率</span>
    </div>
    <!-- 更多 KPI cards -->
  </div>

  <div class="module-grid">
    <div class="module-card">模块A</div>
    <div class="module-card">模块B</div>
    <!-- 4-8 个模块 -->
  </div>

  <div class="summary-flow">
    <!-- SmartArt 流程图 -->
  </div>
</div>
```

## 2. 汇报页

### 特征
- 数据驱动，KPI + 趋势展示
- 图表与文字结合
- 对比强烈（vs 上期、vs 目标）

### 布局结构

```
┌─────────────────────────────────────────────────────┐
│  [L1] 页面标题              [L5] 来源：XX研究院    │
├─────────────────────────────────────────────────────┤
│ ┌───────────────────────┐ ┌───────────────────────┐│
│ │                       │ │  核心发现 1          ││
│ │     图表区域          │ │  核心发现 2          ││
│ │     （占 60% 宽度）   │ │  核心发现 3          ││
│ │                       │ │  核心发现 4          ││
│ └───────────────────────┘ └───────────────────────┘│
├─────────────────────────────────────────────────────┤
│ ┌─────────┬─────────┬─────────┬─────────┐          │
│ │ 指标 1  │ 指标 2  │ 指标 3  │ 指标 4  │          │
│ └─────────┴─────────┴─────────┴─────────┘          │
└─────────────────────────────────────────────────────┘
```

### CSS 模板片段

```html
<div class="page-report">
  <div class="chart-section">
    <img src="chart.png" alt="趋势图" class="chart" />
  </div>

  <div class="insights">
    <div class="insight-item">核心发现 1</div>
    <div class="insight-item">核心发现 2</div>
  </div>

  <div class="metrics-row">
    <div class="metric">
      <span class="metric-value">+23%</span>
      <span class="metric-label">同比增长</span>
    </div>
  </div>
</div>
```

## 3. 架构页

### 特征
- 分层结构，展示系统/组织关系
- 模块化盒子，层级清晰
- 通常有箭头表示数据流或依赖关系

### 布局结构

```
┌─────────────────────────────────────────────────────┐
│  [L1] 架构标题                                      │
├─────────────────────────────────────────────────────┤
│ ┌─────────────────────────────────────────────────┐│
│ │              [Layer 1] 华为红标题               ││
│ │  ┌───────┐ ┌───────┐ ┌───────┐ ┌───────┐       ││
│ │  │ 模块  │ │ 模块  │ │ 模块  │ │ 模块  │       ││
│ │  └───────┘ └───────┘ └───────┘ └───────┘       ││
│ └─────────────────────────────────────────────────┘│
├─────────────────────────────────────────────────────┤
│ ┌─────────────────────────────────────────────────┐│
│ │              [Layer 2] 深灰标题                  ││
│ │  ┌───────┐ ┌───────┐ ┌───────┐ ┌───────┐       ││
│ │  │ 模块  │ │ 模块  │ │ 模块  │ │ 模块  │       ││
│ │  └───────┘ └───────┘ └───────┘ └───────┘       ││
│ └─────────────────────────────────────────────────┘│
├─────────────────────────────────────────────────────┤
│         [Layer 3]  ──────────────────→            │
└─────────────────────────────────────────────────────┘
```

### CSS 模板片段

```html
<div class="page-architecture">
  <h1 class="page-title">系统架构</h1>

  <div class="layer">
    <div class="layer-header">基础设施层</div>
    <div class="module-row">
      <div class="module">计算资源</div>
      <div class="module">存储资源</div>
      <div class="module">网络</div>
    </div>
  </div>

  <div class="layer">
    <div class="layer-header">平台层</div>
    <div class="module-row">
      <div class="module">数据平台</div>
      <div class="module">AI 平台</div>
    </div>
  </div>

  <div class="layer-arrow">▶</div>
</div>
```

## 4. 对比分析页

### 特征
- 双列或多列对比
- 表格+分析结合
- 有明确推荐或结论

### 布局结构

```
┌─────────────────────────────────────────────────────┐
│  [L1] 方案对比分析                                  │
├─────────────────────────────────────────────────────┤
│ ┌───────────────┬───────────────┬───────────────┐  │
│ │    方案 A     │    方案 B     │    方案 C     │  │
│ ├───────────────┼───────────────┼───────────────┤  │
│ │ ✓ 优势 1     │ ✓ 优势 1     │ ✓ 优势 1     │  │
│ │ ✓ 优势 2     │ ✗ 劣势 1     │ ✓ 优势 2     │  │
│ │ ✗ 劣势 1     │ ✓ 优势 3     │ ✗ 劣势 1     │  │
│ └───────────────┴───────────────┴───────────────┘  │
├─────────────────────────────────────────────────────┤
│ ┌─────────────────────────────────────────────────┐│
│ │ ★ 推荐方案 B：性价比最高，适合 XX 场景           ││
│ └─────────────────────────────────────────────────┘│
└─────────────────────────────────────────────────────┘
```

### CSS 模板片段

```html
<div class="page-comparison">
  <table class="comparison-table">
    <thead>
      <tr>
        <th>维度</th>
        <th>方案 A</th>
        <th>方案 B ★</th>
        <th>方案 C</th>
      </tr>
    </thead>
    <tbody>
      <tr>
        <td>成本</td>
        <td>高</td>
        <td class="highlight">中</td>
        <td>低</td>
      </tr>
    </tbody>
  </table>

  <div class="recommendation">
    <span class="star">★</span>
    推荐方案 B：性价比最高
  </div>
</div>
```

## 5. 发布会结论页

### 特征
- 单点突出，信息简洁
- 大字标题 + 关键信息
- 视觉冲击力强

### 布局结构

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│                                                     │
│              [大字] 核心结论 / 核心价值             │
│                                                     │
│                                                     │
├─────────────────────────────────────────────────────┤
│  ┌─────────┐  ┌─────────┐  ┌─────────┐            │
│  │ 关键点1 │  │ 关键点2 │  │ 关键点3 │            │
│  └─────────┘  └─────────┘  └─────────┘            │
└─────────────────────────────────────────────────────┘
```

### CSS 模板片段

```html
<div class="page-conclusion">
  <div class="hero-section">
    <h1 class="hero-title">华为云 AI 解决方案</h1>
    <p class="hero-subtitle">助力企业数字化转型</p>

    <div class="point">高性能计算</div>
    <div class="point">安全可靠</div>

  </div>
</div>
```

## 页型选择指南

| 需求 | 推荐页型 |
|------|---------|
| 全局战略展示 | 战略总览 |
| 周期性工作汇报 | 汇报页 |
| 技术方案设计 | 架构页 |
| 方案选型决策 | 对比分析 |
| 外部演讲发布 | 发布会结论 |
